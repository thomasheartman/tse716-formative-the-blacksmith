An overview over the technical equipment I used, including DAW, key synths and plugins, and more. For brevity, I have omitted anything that Guy uses in his composing videos. I also wrote my version in Logic, so all the same synths and plugins that he uses, I have also used.

\begin{itemize}
  \item \textbf{DAW:} Logic Pro 11.1.2

  \item \textbf{Libraries:}
  \begin{itemize}
    \item EastWest Hollywood Fantasy Strings (Dulcimers)
    \item EastWest Stormdrum 2 (Taikos, Middle Earth ensemble)
  \end{itemize}

  \item \textbf{Loops:}
  \begin{itemize}
    \item Clock Sync Beat 01 (Apple Loops)
  \end{itemize}

  \item \textbf{Plugins:\footnote{This list is not exhaustive, but contains the plugins that I consider to be the most important for the track. I spoke to Joel Grant about this in a 121, and he told me (and I agree) that an exhaustive list of plugins isn't useful to anyone.}}
  \begin{itemize}
    \item Cluster Delay (Minimal Audio)
    \item Neutron 5 (Izotope)
    \item OTT (Xfer records)
    \item Ozone 11 (Izotope)
    \item Shaperbox 3 (Cableguys)
    \item Snapback (Cableguys)
    \item kHs Trance gate (Kilohearts)
  \end{itemize}

  \item \textbf{Synths and other instruments:}
  \begin{itemize}
    \item Life (XLN Audio)
    \item Pigments (Arturia)
    \item Random (Beatsurfing)
    \item Random Metal (Beatsurfing)
    \item Serum 2 (Xfer records)
  \end{itemize}

\end{itemize}
